# Security Policy

## Supported Versions

| Version   | Supported |
|-----------|-----------|
| 0.1.x     | Yes       |

## Reporting a Vulnerability

Please report suspected security vulnerabilities via
[GitHub Security Advisories](https://github.com/anycompany-one/kagarr/security/advisories/new).

You will receive a response within 72 hours. If the issue is confirmed,
a fix will be released as soon as possible depending on severity.

Please do **not** open public issues for security vulnerabilities.
